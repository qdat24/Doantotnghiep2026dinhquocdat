# Logo Website

Đặt file logo của bạn vào thư mục này với tên `logo.png`

Logo sẽ được tự động hiển thị trên website nếu:
- Không có logo được cấu hình trong Admin Settings
- File logo.png tồn tại trong thư mục này

Kích thước khuyến nghị: 
- Chiều cao: 60px
- Định dạng: PNG (nền trong suốt) hoặc SVG
- Tỷ lệ: Tự do, nhưng nên giữ chiều cao khoảng 60px
